===============================================================================
                    PART 1: BEFORE YOUR PRESENTATION (SETUP)
===============================================================================

## STEP 1: Open Command Prompt / PowerShell

Option A: Press Windows + R, type "cmd", press Enter
Option B: Press Windows + X, select "Windows Terminal" or "PowerShell"
Option C: Search "PowerShell" in Start Menu

-------------------------------------------------------------------------------

## STEP 2: Navigate to Your Project

Type this command and press Enter:

    cd C:\Projects\intelli-migrate\backend

You should see:
    C:\Projects\intelli-migrate\backend>

-------------------------------------------------------------------------------

## STEP 3: Activate the Python Virtual Environment

Type this command and press Enter:

    .\venv\Scripts\Activate.ps1

You should see "(venv)" appear at the start of your prompt:
    (venv) C:\Projects\intelli-migrate\backend>

If you get an error about "execution policy", run this first:
    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Then try the activate command again.

-------------------------------------------------------------------------------

## STEP 4: Start the Backend Server

Type this command and press Enter:

    uvicorn main:app --reload --port 8000

You should see:
    INFO:     Uvicorn running on http://127.0.0.1:8000
    INFO:     Started reloader process...
    ✅ NLP Model loaded: all-MiniLM-L6-v2
    ✅ Anomaly Detection Model loaded: IsolationForest

⚠️ KEEP THIS WINDOW OPEN! Don't close it. The server must stay running.

-------------------------------------------------------------------------------

## STEP 5: Open a NEW Terminal Window (for frontend)

- Right-click on PowerShell in taskbar → "Windows PowerShell"
- Or press Windows + R, type "cmd", press Enter

In the NEW window, type:

    cd C:\Projects\intelli-migrate\frontend
    python -m http.server 3000

You should see:
    Serving HTTP on :: port 3000 (http://[::]:3000/) ...

⚠️ KEEP THIS WINDOW OPEN TOO!

-------------------------------------------------------------------------------

## STEP 6: Open Your Platform in Browser

Open Google Chrome (or any browser) and go to:

    http://localhost:3000

🎉 YOUR PLATFORM IS NOW LIVE!

===============================================================================
                    PART 5: TROUBLESHOOTING
===============================================================================

PROBLEM: "uvicorn is not recognized"
SOLUTION: Make sure you activated the virtual environment:
    .\venv\Scripts\Activate.ps1

PROBLEM: "Port 8000 already in use"
SOLUTION: Use a different port:
    uvicorn main:app --reload --port 8001
    (Then update API_BASE in frontend/index.html)

PROBLEM: "Frontend shows 'API Offline'"
SOLUTION: Make sure the backend server is running in the other terminal

PROBLEM: "First run is slow"
SOLUTION: First run downloads the NLP model (~90MB). This is normal.
          Subsequent runs will be fast.

PROBLEM: "PowerShell script execution error"
SOLUTION: Run this command first:
    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
